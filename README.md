## 🚀 Quick start

**Run locally**

```shell
gatsby develop
```

## 💫 Deploy

Using the gh-pages package

```shell
npm install gh-pages --save-dev
{
    "scripts": {
        ...
        "deploy": "gatsby build && gh-pages -d master-source -b master",
    }
}
npm run deploy
```
